Legislative Transparency Platform

A comprehensive full-stack application that automates Congressional bill tracking with AI-powered summaries, making legislative information accessible for civic engagement initiatives. The system processes thousands of bills daily, generating intelligent summaries and enabling quality assurance through expert feedback collection.

Technologies Used
Python Flask PostgreSQL SQLAlchemy Claude API Congress.gov API Railway REST APIs
Key Features
  • Automated daily data ingestion from Congress.gov API with action-type filtering
  • AI-powered bill summarization using Claude API with two-tier content formatting
  • Complex database schema managing thousands of legislative records with full relationship modeling
  • Quality assurance pipeline with multi-dimensional expert feedback for continuous AI optimization
  • Admin dashboards for API usage tracking and cost monitoring
